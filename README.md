# Destino Firme Empreendimentos

Landing page premium em **Next.js 15 + TypeScript + Tailwind 3** para a Destino Firme Empreendimentos — remodelações, pladur, pintura, barramentos e acabamentos.

## Stack

- Next.js 15 (App Router)
- React 19 RC
- TypeScript
- Tailwind CSS 3
- Lucide icons
- Fontes: Manrope (display) + Inter (body)

## Desenvolvimento

```bash
npm install
npm run dev    # http://localhost:3000
npm run build
npm run start
```

## Estrutura

```
src/
├── app/                # App Router (layout, page, globals)
├── components/         # Header, Hero, Services, etc.
└── lib/                # constants.ts, scrollReveal.ts
public/
└── logo.png
```
